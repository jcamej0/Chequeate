== Description ==

Here comes Tavisha. All the content you post online will now be beautifully and perfectly laid out on your home page, thanks to Tavisha you can choose a layout will fit for your site (e.g. two-column left-sidebar or two-column right-sidebar).

Tavisha is clean, fluid and responsive, making your website look great no matter on what kind of device or screen size it's being viewed. Whether your visitors access your website on a desktop, an iPad or a smartphone, content is always delivered with ultimate focus.

* Responsive Layout
* Custom Colors
* Custom Layout
* Menu Description
* Social Links
* The GPL v2.0 or later license. :) Use it to make something cool.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's ZIP file. Click Install Now.
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==

= How do I change the color scheme? =

You can change the colors of your site easily using Tavisha.

1. In your admin panel, go to Appearance -> Customize.
2. Now you will see the Customizer and a tab called 'Colors'. Click this tab.
3. Should you wish to create your own color scheme, you can by selecting the colors for each area listed.
4. Once you are happy with your color changes you can click save and your changes will be reflected on your live site.

= How do I change the sidebar layout? =

Tavisha allow you to change sidebar layout between Sidebar on Right or Sidebar on Left

1. In your admin panel, go to Appearance -> Customize.
2. Now you will see the Customizer and a tab called 'Layout Setting'. Click this tab.
3. You can now change your sidebar layout by choosing one of the predefined ones. You can preview the change in the Customizer.
4. Once you are happy with your layout changes you can click save and your changes will be reflected on your live site.

= How do I add the Social Links to the footer? =

Tavisha allows you display links to your social media profiles, like Twitter and Facebook, with icons.

1. Create a new Custom Menu, and assign it to the Social Links Menu location.
2. Add links to each of your social services using the Links panel.
3. Icons for your social links will automatically appear if it's available.

= How do I add a description for my menu link in navigation? =

Tavisha a menu design that's easy to navigate -- especially when you add menu descriptions.

1. Visit the Menus page in your admin.
2. Use the Screen Options tab to "Show advanced menu properties".
3. Select "Description" there to start editing menu descriptions.
4. Select the menu you want to add links and descriptions to.
5. When in the Menu Structure section, you can click open the link and add a description.
6. Once you save the menu with your link, the new description should show up.
